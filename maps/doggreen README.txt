TITLE                   : Omaha Beach: Dog Green Sector
FILENAME                : doggreen.zip
BSP NAME		: doggreen.bsp
AUTHOR                  : Christian Howland A.K.A. The Violator
RELEASE DATE            : 6\12\05


This was a project I worked on for about a year on and off. I wasn't fully satisfied with any of 
the D-Day beach maps out there so I decided to make my own. I purposely made the map
challenging for the allies so that it would require a great deal of teamwork to win. 
Have Fun !

DOWNLOADS:

What you NEED to run this map:

dday 4.0 -  
http://dynamic.gamespy.com/~dday/site/files.php

-IMPORTANT-
In order for the secondary spawn doors for the allies to operate and for the turrets to
work, you need SHAEF, available at http://www.planetquake.com/ddc
